# Client Logiflow ERP

## ✅ Iniciar solo el host:
npm run dev:host

## ✅ Iniciar solo el remoto (logistics):
npm run dev:remote

## ✅ Iniciar ambos (host + remoto):
npm run dev:all

## ✅ Construir todo:
npm run build

## ✅ Servir en modo producción:
npm run serve

✅ Detener puertos ocupados:
npm run stop
